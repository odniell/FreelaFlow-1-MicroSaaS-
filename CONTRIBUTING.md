# 🤝 Como Contribuir

Obrigado pelo interesse em contribuir com o FreelaFlow!

## Processo

1. **Fork** o repositório
2. **Clone** seu fork: `git clone https://github.com/SEU-USUARIO/freelaflow.git`
3. **Crie uma branch** para sua feature: `git checkout -b feature/nome-da-feature`
4. **Faça suas mudanças** e teste no navegador
5. **Commit** com mensagem clara: `git commit -m 'feat: descrição da mudança'`
6. **Push**: `git push origin feature/nome-da-feature`
7. **Abra um Pull Request** descrevendo o que foi alterado

## Convenções de Commit

Seguimos [Conventional Commits](https://conventionalcommits.org):

- `feat:` nova funcionalidade
- `fix:` correção de bug
- `docs:` mudança na documentação
- `style:` mudança visual/CSS
- `refactor:` refatoração de código
- `perf:` melhoria de performance

## Reportar Bugs

Abra uma [Issue](https://github.com/SEU-USUARIO/freelaflow/issues) com:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Screenshot (se aplicável)
- Navegador e sistema operacional

## Sugerir Features

Abra uma Issue com o label `enhancement` descrevendo:
- Qual problema resolve
- Como deveria funcionar
- Mockup ou exemplo (opcional)
