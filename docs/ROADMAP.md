# 🗺️ Roadmap FreelaFlow

## v1.0.0 — MVP ✅ (Atual)
- [x] Landing page completa com conversão
- [x] Sistema de login/cadastro (frontend)
- [x] Dashboard com métricas e gráficos
- [x] CRUD completo de clientes
- [x] Cobrança via WhatsApp
- [x] Calculadora inteligente de preços
- [x] Sistema de planos Free/Plus (frontend)
- [x] Alertas de pagamentos atrasados
- [x] Design responsivo

---

## v1.1.0 — Persistência Local 🔜
- [ ] Salvar dados no `localStorage`
- [ ] Exportar clientes para CSV
- [ ] Importar clientes via CSV
- [ ] Modo escuro (dark mode)
- [ ] Filtros avançados na tabela de clientes

## v1.2.0 — Backend com Supabase 🔜
- [ ] Autenticação real (email/senha + Google OAuth)
- [ ] Banco de dados PostgreSQL via Supabase
- [ ] Dados sincronizados entre dispositivos
- [ ] Multi-usuário
- [ ] Backup automático

## v1.3.0 — Pagamentos 🔜
- [ ] Integração Abacate Pay
- [ ] Assinatura mensal automática (R$29/mês)
- [ ] Upgrade/downgrade de plano
- [ ] Histórico de faturas
- [ ] Webhook de confirmação de pagamento

## v1.4.0 — Comunicação 🔜
- [ ] Templates de cobrança personalizáveis
- [ ] Agendamento de cobranças automáticas
- [ ] Notificações por e-mail (vencimentos)
- [ ] Histórico de mensagens enviadas

## v2.0.0 — Plataforma Completa 🔮
- [ ] PWA (Progressive Web App) — instalar no celular
- [ ] Geração de contratos simples (PDF)
- [ ] Emissão de recibos/notas
- [ ] Integração com Google Calendar (prazos)
- [ ] Relatórios avançados com export PDF
- [ ] API pública para integrações
- [ ] Programa de afiliados
